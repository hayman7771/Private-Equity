# LBO Model Specification – MIS

## Core Components

### 1. Sources & Uses
- Sponsor equity
- Term loan
- Rollover equity (if applicable)
- Transaction fees
- Sources must equal uses

### 2. Operating Model
- Revenue build
- Margin expansion roadmap
- Depreciation & amortization
- Working capital schedule
- Capex schedule

### 3. Debt Schedule
- Opening balances
- Interest expense
- Mandatory amortization
- Excess cash sweep

### 4. Cash Flow Waterfall
EBITDA  
– Capex  
– ?NWC  
– Cash taxes  
= Unlevered FCF ? Available for debt paydown

### 5. Returns Analysis
- 5-year exit
- Exit multiple assumption
- IRR
- MOIC
- Sensitivity table (entry/exit multiples, leverage levels)

This spec mirrors middle-market sponsor models.
